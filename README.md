2016-4-10 更新了RelatedFunctionv1.5
1.添加了GoalCharacterd的属性legal
	用于表示离散关联函数用户维护的属性值是否都为正值或都为负值
	如果都为正值，则legal为true，当属性值不在维护的属性值中，则返回-100；
	如果都为负值，则legal为false，当属性值不在维护的属性值中，则返回100.
